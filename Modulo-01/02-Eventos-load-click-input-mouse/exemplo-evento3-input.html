Log de ocorrências:



onBlur -> Quando sai do campo.

onChange -> Quando muda o conteúdo do campo.

onFocus -> Quando entra no campo.

onSelect -> Quando um texto é selecionado.

onSubmit -> Quando o formulário é submetido. Antes de chamar a função, precisa colocar return, por exemplo: "return nomedafuncao()". Pois ele normalmente fazemos aqui algumas verificações das informações preenchidos no form. Ou seja, ele só envia as iformações se tudo estiver validado.

onReset -> Quando o botão de reset é clicado.

onKeyDown -> Quando pressionada a tecla.

onKeyUp -> Quando solta a tecla pressionada.
