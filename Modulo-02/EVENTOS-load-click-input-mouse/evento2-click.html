Eventos de Click

Ações serão executadas após Eventos de Click (onClick e onDblClick) serem carregados.



Exemplo:



Log de ocorrências:



onClick -> Executa quando um elemento é clicado uma vez.

onDblClick -> Executa quando um elemento é clicado duas vezes (clique rápido).

getElementById -> Procura algum elemento da página com este id.

innerHTML -> Introduz um código html.



Código js:

function umClique() {
document.getElementById("imagem").src="img/imagem-exemplo-02.jpg";
document.getElementById("log").innerHTML += "Troca para a segunda imagem.<br>";
}

function doisCliques() {
document.getElementById("imagem").src="img/imagem-exemplo-01.jpg";
document.getElementById("log").innerHTML += "Troca para a primeira imagem.<br>";
}

Código html:

<img src="img/imagem-exemplo-01.jpg" id="imagem" onClick="umClique()" onDblClick="doisCliques()" width="300">
