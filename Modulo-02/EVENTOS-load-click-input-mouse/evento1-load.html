1Eventos de Load

Ações serão executadas após Eventos de Load (onLoad e onResize) serem carregados.



Log de ocorrências:



document -> Significa a página html onde o arquivo .js está sendo chamado. Por exemplo, aqui, o document significa exemplo-evento1-load.html

getElementById -> Procura algum elemento da página com este id.

innerHTML -> Introduz um código html.



Código js:

function carreguei() {
document.getElementById("log").innerHTML += "Carreguei!⁢*br*>";
}

function redimensionei() {
document.getElementById("log").innerHTML += "Redimensionei!⁢*br*>";
}
